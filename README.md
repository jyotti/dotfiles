# dotfiles

## Overview

- Shell: Bash
- Terminal: tmux, [Powerline](https://github.com/powerline/powerline)

## Installation

### Prerequisites

- Command Line Tools (CLT) for Xcode `xcode-select --install`
- [Homebrew](https://brew.sh/)

### Install formula and dotfiles

Install formula with Hombrew-bundle

```bash
brew bundle
```

Install dotfiles

```bash
bash bootstrap.sh
```
